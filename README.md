Schnelleinstieg Banana Pi
======================

In diesem GitHub-Repository finden Sie weiterführende Infos, Codeschnippsel
und Schaltskizzen im Fritzing-Format als Begleitmaterialien zu meinem Buch
"Schnelleinstieg Banana Pi", dessen erste Auflage im Juni 2015 im Franzis
Verlag erschienen ist.

 * [Eintrag in der DNB](http://d-nb.info/1066390452)
 * [Amazon-Bestellseite](http://www.amazon.de/gp/product/3645603905/ref=as_li_tl?ie=UTF8&camp=1638&creative=19454&creativeASIN=3645603905&linkCode=as2&tag=mattiasschlen-21&linkId=TS2NKRFHCBHPMEE5)
 
Sollten Sie mit git und GitHub nicht vertraut sein, klicken Sie einfach auf 
"[Download Zip](https://github.com/mschlenker/Schnelleinstieg_Banana_Pi/archive/master.zip)" 
rechts. Für den

## Formate

 * Bilder liegen als PNG und JPG vor, hoch aufgelöst und oft nicht
 nachbearbeitet - das hilft, Kabelwege zu erkennen, steigert aber Ladezeiten
 * Schaltskizzen im .fzz-Format öffnen Sie mit 
 [Fritzing](http://fritzing.org/download/) - da können Sie zoomen, Bauteile
 verschieben und sogar Arduino-Shields designen
